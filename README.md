# 🌟 My Bucket List Web App

## 📌 Project Overview

This project is an interactive **Bucket List web application** built using HTML, CSS, and JavaScript. It allows users to create, manage, and track their personal goals or dreams in a simple and visually appealing interface.

The application updates dynamically without reloading the page and stores data locally so users do not lose their progress.

---

## 🚀 Features

* ➕ Add new bucket list items
* ✅ Mark items as completed
* ❌ Delete individual items
* 🔍 Filter items (All / To Do / Completed)
* 🧹 Clear completed items or entire list
* 📊 Live statistics (total, completed, percentage)
* 📈 Progress bar showing completion status
* 💾 Data persistence using **localStorage**

---

## 🧠 Technologies Used

* **HTML5** – Structure of the web page
* **CSS3** – Styling, layout, and animations
* **JavaScript (ES6)** – Logic, DOM manipulation, and interactivity
* **localStorage (Option A)** – Saving user data between sessions

---

## ⚙️ How It Works

1. The user enters a goal in the input field and clicks **Add**
2. JavaScript captures the event using event listeners
3. The item is added to the list and displayed dynamically
4. Data is saved to **localStorage**
5. When the page reloads, saved data is retrieved and displayed
